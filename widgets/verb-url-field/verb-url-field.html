URL control (standalone widget)

With a configured base list, a path-only value (starts with /) shows the dark orange {base} label; copy and open resolve using the default base (first in the list) plus your path. On / or /Pat (only the first path segment, before a second /), an alphabetical FHIR R4 resource typeahead opens. Deeper paths such as /Patient/123 do not use that list. For non-path values, type substrings of a configured base URL to open the base URL list. Single-click the copy icon for the resolved URL; double-click to open it in a new tab.
