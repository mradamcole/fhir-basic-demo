Manual checklist

  1. In the iframe, type localhost (or another substring of a configured base) — base URL list should appear (typeahead).
  2. Type only / — {base} shows for path-only mode and an alphabetical FHIR resource list should open; pick one (e.g. Patient) to set /Patient.
  3. Type /Patient/1 — dropdown should not offer the full FHIR resource list (only first segment uses it); /local should fall back to base URL matches when no resource name matches.
  4. Pick a base from the base URL list — value should become that base URL (or joined with path when the value was path-only).
  5. Single-click the copy icon — URL copied; icon briefly shows a checkmark.
  6. Double-click the copy icon — should open the resolved URL in a new tab (path-only values use the first configured base).
  7. With a path like /Patient, a dark orange {base} prefix should appear before the field; copy should include the full URL.
